### 插件特色
#### 主插件DRDK v1.9.2插件（最没存在感的插件，很容易被忽略）
罕见的选择了包抗为主的高级对抗，调整了大多数的参数使得生还者进屋没有那么容易

#### 副插件Evil Hunter v1.6.3插件（好家伙，这插件才是主要的）
罕见的将WP与包抗进行融合的诡异对抗，虽然进屋很容易，但拿高分并不是很简单

别的模式都不重要，直接略过
